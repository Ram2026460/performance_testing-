1.What is performance testing ? 
Performance Testing is a type of software testing that checks how fast, stable, and scalable an application is under a specific workload.

*to test scalability
*to increase user experience
*to avoid production failure 

Diff Types Of testing ?
Load Testing – Expected number of users
Stress Testing – Beyond system capacity
Spike Testing – Sudden traffic increase
Soak (Endurance) Testing – Long duration load
Scalability Testing – System growth capability
Volume Testing – Large data handling

PT is usually measured In?
RESPONSE TIME :time taken for a request to get response
THROUGHPUT :number of request`s handled per second
TPS :no; of transections done per second 
LATENCY: delay between sending request and receiving response
ERROR RATE: percentage of failed requests
